# **App Name**: FoodDash

## Core Features:

- Restaurant Listing: Display a list of restaurants based on user location and search criteria.
- Menu Display: Show the menu items for a selected restaurant.
- Order Placement: Allow users to add items to a cart and place an order.
- User Authentication: Enable user registration, login, and profile management.
- Order Tracking: Provide real-time order status updates to the user.
- Restaurant Search: Users can search for restaurant names.
- AI-Powered Recommendations: Use a tool to generate restaurant recommendations based on user preferences and past order history.

## Style Guidelines:

- Primary color: Vibrant orange (#FF7043) to convey energy and appetite.
- Background color: Light orange (#FFDAB9) for a warm and inviting feel, derived from the primary hue but desaturated to 25% saturation.
- Accent color: Yellow-orange (#FF9800), analogous to the primary color, providing contrast for calls to action.
- Font: 'Poppins', a geometric sans-serif font for a clean and modern look suitable for short blocks of text, headlines, and user interface elements.
- Use clear, minimalist icons for categories, menu items, and user actions.
- Clean and responsive layout with a focus on mobile-first design.
- Subtle animations for transitions and user feedback to enhance the user experience.
